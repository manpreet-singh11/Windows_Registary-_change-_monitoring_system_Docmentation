# Registry Security Monitor

A Python-based security tool designed to detect unauthorized or malicious changes to the Windows Registry in real-time.

## ✨ Key Features
- **Real-time Monitoring:** Tracks additions, modifications, and deletions.
- **Risk Categorization:** Automatically flags events as Low, Medium, High, or Critical.
- **Security Intelligence:** Detects common malware tactics like disabling Windows Defender.
- **Audit Reporting:** Generates formatted text reports with data analysis using Pandas.
